
KEYBOARD SHORTCUTS
For all short cuts below make command to ctrl for windows.
Copy 
MAC -> Command + C
WINDOWS -> CTRL + C

Paste
MAC -> Command + V

Select All -> Command + A 

Cut -> Command + X 

Revert Back -> Command + Z 

New -> Command + N

Save -> Command + S

MAC -> Screen shot short cut is 
command + shift + 4 

WINDOWS Screen shot-> Windows + Shift + S
